# 🎰 TFT Roll Calculator (Set 17)

An interactive, hyper-accurate rolling calculator for **Teamfight Tactics (Set 17: Space Gods)**. This tool uses a dynamic JavaScript Monte Carlo simulation (10,000 iterations) to accurately predict your odds of hitting specific unit counts while factoring in shared champion pool depletion in real-time.

## 🛠️ Features
- **Accurate Shared Pool Math:** Simulates actual TFT bag sizes and pool shrinking as players hold units.
- **Set 18 Future-Proofing:** Built with a scalable data architectural layer to easily switch parameters when the next set drops.
- **Gold Variance Table:** Displays success probabilities across `+/- 10` and `20` gold thresholds to optimize your economy decisions.
- **Zero Interface Clutter:** No manually typing total champion roster sizes; everything is configured cleanly in the backend.
